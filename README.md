# CarNotes

### Implemented:
- Login, Register Pages
- JWT (JSON Web Token) auth
- Garage Page, notes counter (does not count spare parts, actions only)
- Car profile Page
- Mileage chart
- CRUD actions for Refuelings, Washings, Spare Parts, Services, Legal Procedures
- Neo4j data storage
- REST API
